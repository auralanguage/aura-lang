# Contributing to Aura

First off, thank you for considering contributing to Aura!

## Important Note
Aura is currently in **pre-alpha**. The API, syntax, and toolchain are still evolving. For significant changes, please open an issue first to discuss them.

## 🐛 Reporting Bugs
- Check **existing issues** first (your bug might already be reported)
- Open a new **issue** with the following information:
  - Operating system version (Windows 10/11?)
  - Aura version
  - Minimal code example that reproduces the bug
  - Expected vs. actual behavior

 ## Suggesting Features
- Check **existing issues** first (maybe someone already suggested it)
- Open a new **issue** with `[Feature]` in the title
- Describe the feature and why it's needed

## Development
**Run tests (make sure your changes don't break anything)**
```bash
.\build\aura.exe test --manifest tests/test_cases.json --executable build/aura.exe
```
## Questions?
- Open an issue for any questions
- For major changes, always open an issue first to discuss
